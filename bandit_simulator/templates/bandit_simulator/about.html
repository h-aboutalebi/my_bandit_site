Our Mission

{# introduction paragraph#}

This project is part of the Quebec SPOR SUPPORT Unit to enhance patient monitoring and improve primary care. The main part of this process contains decision making based on a given constraints and demands.
Here, we try to address part of this problem by utilizing machine learning algorithms and help practitioners better understand the efficacy of these models. Considering the fact that many research papers provide limited illustrations of their work for different problem settings, we aim to provide a tool kit for practitioners which can extensively study different problem settings and can further help practitioners in model selection.

Our Team
